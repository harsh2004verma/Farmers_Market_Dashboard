# 🌾 Farmers Market Dashboard

## 📌 Project Overview

The Farmers Market Dashboard is an interactive Power BI report designed to analyze sales, quantity, customers, vendors, and product performance within a farmers market environment.

This dashboard enables users to monitor key business metrics, identify top-performing products, evaluate customer purchasing behavior, and gain insights into vendor performance through dynamic visualizations and filters.

---

## 🎯 Objectives

- Track overall sales and quantity performance.
- Analyze customer purchasing patterns.
- Compare vendor contributions.
- Monitor product-level sales and quantity.
- Explore customer demographics using ZIP code analysis.
- Provide interactive filtering for detailed business insights.

---

## 🛠️ Tech Stack

The dashboard was built using:

- **Power BI Desktop** – Data visualization and reporting.
- **Power Query** – Data cleaning and transformation.
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs.
- **Data Modeling** – Establishing relationships between tables.
- **Excel / CSV Dataset** – Source data for analysis.

## 📊 Dashboard Pages

### 1️⃣ Combined KPI Analysis

This page provides an overview of the most important business metrics:

#### Key Performance Indicators (KPIs)

- Total Quantity
- Unique Customers
- Total Customers
- Total Vendors
- Total Booths

#### Visualizations

- Product Name vs Quantity
- Vendor vs Quantity
- Customer vs Quantity
- Customer ZIP vs Quantity
- Product vs Quantity
- Customer Quantity Detail Table

#### Features

- Interactive Filters:
  - Customer
  - Vendor
  - Date
  - Product
